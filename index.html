General Information

This workshop is hosted by the SIB Swiss Institute of Bioinformatics and the Institute of Molecular Life Sciences at the University of Zurich (UZH).

Data Carpentry workshops are for any researcher who has data they want to analyze, and no prior computational experience is required. This hands-on workshop teaches basic concepts, skills and tools for working more effectively with data.

We will cover {{page.day1_am}}, {{page.day1_pm}}, {{page.day2_am}} and {{page.day2_pm}}. Participants should bring their laptops and plan to participate actively. By the end of the workshop learners should be able to more effectively manage and analyze data and be able to apply the tools and approaches directly to their ongoing research.

Who: The course is aimed at graduate students and other researchers with little to no R experience.

Lesson content

Course materials will be based on the Data Carpentry Ecology Workshop and Genomics Workshop lessons, available on the Data Carpentry website.

  • Introduction to R; Data analysis in R: These lessons will be based on the R-ecology: Data Analysis and Visualization in R and R-genomics: Data analysis in R lessons. You will need to download the following files: combined.csv (for the R-ecology lesson) and Ecoli_metadata.csv (for the R-genomics lesson). Links to these files are also available in the lesson materials.
  • Data organization in spreadsheets: This session will show you best practices for organizing experiment meta-data in spreadsheets. The materials will be based on the Data Carpentry Spreadsheets for Ecology lesson. Please download the following spreadsheet file for this lesson: survey_data_spreadsheet_messy.csv.
  • Data visualization in R: This session will introduce plotting in R using the ggplot2 framework. Lesson materials are available here: https://github.com/jennybc/ggplot2-tutorial.
  • Introduction to the command line: This session will be based on the Software Carpentry Unix Shell lesson. You will need a working Terminal application for the exercises. On Mac and Linux systems, this is installed by default (for Mac users: try searching for "Terminal" in Spotlight Search, or look for Terminal.app under Utilities in Applications). Windows users should install Git Bash. If you are using a Windows laptop, please download, install, and try to open the application before Tuesday afternoon.

Setup

To participate in a Data Carpentry workshop, you will need working copies of the described software. Please make sure to install everything (or at least to download the installers) before the start of your workshop. Participants should bring and use their own laptops to insure the proper setup of tools for an efficient workflow once you leave the workshop.
